## 其他版本

**Electron App** 是主要维护的版本，无论是 docker 部署、Node 源码运行。感觉用起来都很麻烦，所以想做一个傻瓜式的应用，只需要点点点，就完事了。

NodeJs 代码运行，docker 部署。可以转到 [main 分支](https://github.com/rao-pics/core/tree/main)
