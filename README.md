# <img src='https://raw.githubusercontent.com/rao-pics/rua/main/public/favicon.ico?raw=true' height="24px" width="24px" /> Rao.Pics

基于 Eagle/Pixcall/Billfish 等素材管理工具，用于局域网内**展示**已有的图库。

## ✈️ 超级简单

```
pnpm i @raopics/use
```

```js
// index.mjs

import EagleUse from "@raopics/use";

EagleUse({
  library: "xxx",
});
```
