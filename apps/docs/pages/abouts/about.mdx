## 项目结构

Fork: [create-t3-turbo](https://github.com/t3-oss/create-t3-turbo)。

```sh
# App 相关

- apps
  - electron                    # electron
  - docs                        # 官网

# 核心方法

- packages
  - api                         # trpc router & trpc-openapi
  - constant                    # 常量
  - db                          # 数据库相关内容
  - eagle                       # 处理 eagle 数据转换为 sqlite
    - watch                     # 对资源库进行监听
  - static-server               # 静态资源服务 => 通过 http 访问本地资源
  - utils                       # 工具类方法

# 开发环境相关

- tooling
  - eslint
  - prettier
  - tailwind                    # tailwind + daisyui
  - typescript

# 快速生成 package 模板

- turbo/*
```
