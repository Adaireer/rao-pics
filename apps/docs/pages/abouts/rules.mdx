# 规范

记录需求、开发、UI、测试方面的一些规范和想法，应对头脑发热的情况。

- 弹窗：使用系统内部弹窗
- 线上错误使用 `Sentry` 收集，先本地 showErrorMesssage，再上报

## i18n

在每个页面中，单独定义一个 `i18n` 对象，用于存放当前页面的国际化文案，而不是使用一个全局的 `i18n` 对象。

当同步应用修改语言时，在 electron renderer 中监听 `language-changed` 事件，然后切换语言即可。

```sh
- page1
    - i18n.js
- page2
    - i18n.js
- page3
    - i18n.js
```
